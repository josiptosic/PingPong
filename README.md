PingPong - posljednja verzija - 1.0.2
Aplikacija je izvedena kao C++ projekt u Visual Studiju 2019, u Windows 10 operacijskom sustavu.
Korišten je ugraðeni VC++ x64 Debugger.

Vizualizacije su napravljene uz pomoæ SDL2 biblioteke.
	Verzija biblioteke je 2.0.20 preuzeta sa službene stranice:
		https://www.libsdl.org/release/SDL2-devel-2.0.20-VC.zip

	Na standardnu biblioteku nadodano je i proširenje za iscrtavanje teksta uz odabrani TTF font,
		preuzeto sa pripadajuæeg git repozitorija:
		https://github.com/libsdl-org/SDL_ttf/releases/download/release-2.0.18/SDL2_ttf-devel-2.0.18-VC.zip

Biblioteke su u projektu uvezane linkerima.
